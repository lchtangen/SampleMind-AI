## 📞 Support & Troubleshooting

### **If Tests Fail:**
1. Read error message carefully
2. Check test fixture setup
3. Verify database migrations applied
4. Run single test in isolation
5. Check for race conditions
6. Consult test-runner agent

### **If CI Fails:**
1. Run `/check` locally
2. Fix first error shown
3. Commit fix
4. Push and verify
5. Repeat if needed

### **If Performance Target Missed:**
1. Profile with `time` command
2. Check for N+1 queries
3. Verify indexes exist
4. Use batch operations
5. Consult audio-analyzer agent

### **If Build Fails:**
1. Check Rust version
2. Verify dependencies installed
3. Clean build: `cargo clean`
4. Rebuild: `pnpm tauri build`
5. Consult tauri-builder agent

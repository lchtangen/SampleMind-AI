## Production Notes

### Recording Setup

1. **Screen Recording**:
   - Use OBS Studio or QuickTime
   - 1920x1080 resolution
   - 30fps
   - Clean desktop background

2. **Terminal Setup**:
   - Font: JetBrains Mono or Fira Code
   - Size: 16-18pt (readable)
   - Theme: Cyberpunk (default)
   - Window: Full screen or large window

3. **Audio**:
   - Voiceover: Clear, professional microphone
   - Background music: Low volume (10-15%)
   - Sound effects: Subtle (optional)

### Editing

1. **Cuts**:
   - Keep it fast-paced
   - No dead air
   - Smooth transitions

2. **Annotations**:
   - Highlight important text
   - Arrow callouts for key features
   - Text overlays for emphasis

3. **Color Grading**:
   - Enhance terminal colors
   - Consistent brightness
   - Professional look

### Export Settings

- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080
- **Bitrate**: 8-10 Mbps
- **Audio**: AAC, 192 kbps

### Distribution

Upload to:
- YouTube (main channel)
- Twitter/X (30-second teaser)
- LinkedIn (full video)
- GitHub README (embedded)
- Discord (announcement)

### Thumbnail

Create eye-catching thumbnail:
- SampleMind AI logo
